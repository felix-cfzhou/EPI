# Know Your Primitive Types

- Bitwise Operators
    - `&` bitwise AND
    - `|` bitwise OR
    - `>>`, `<<` bitwise shift
    - `~` bitwise NOT
    - `^` bitwise XOR
- Bitmasks
- clear lowermost set bit
    - get index of lowermost set bit
    - set lowermost 0
    - etc
- signedness + shifting
- caching small inputs
- use commutativity, associativity to perform parallel computations and reordered computations
